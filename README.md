# uv-package-publisher-skill
A comprehensive workflow guide for building and releasing Python packages using modern `uv` tooling. 

## Capabilities
- **Project Initialization**: Modern setup with `uv init` and `uv add`.
- **Architecture**: Guidance on single-package vs. monorepo (workspace) layouts.
- **Distribution**: Step-by-step publishing to TestPyPI and PyPI.
- **Configuration**: Best practices for `pyproject.toml`, entry points, and `uv.lock`.

Use this skill when converting scripts to tools, setting up MCP servers, or managing multi-package dependencies.
